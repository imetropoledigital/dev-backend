# dev-backend

Projeto de exemplo para a disciplina de **Desenvolvimento Backend** do IMD/UFRN.

## Pré-requisitos

- [Node.js](https://nodejs.org/) v14+
- npm

## Instalação

```bash
npm install
```

## Execução

```bash
# Execução direta
npm start

# Modo desenvolvimento com hot-reload (nodemon)
npm run dev
```
